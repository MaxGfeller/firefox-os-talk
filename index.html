Architecture

Gaia
The user interface for Firefox OS. Homescreen, Lockscreen etc. Completely written in HTML, CSS, JavaScript. Only interface to the Gecko layer are open Web APIs.
Gecko
Networking stack, graphics stack, layout engine, JavaScript virtual machine etc.
Gonk
Linux Kernel, Hardware Abstraction Layer (HAL)

Packaged Apps and Open Web Apps

type: privileged

Use of certain APIs, approved by an app store, resources are signed by the app store

type: certified

Critical system function like dialer, or system settings app. Must be approved by carrier or OEM

type: web

Packaged web sites, no use of special APIs

Open Web APIs 2/2

Example:
var contact = new mozContact();
contact.init({name: "John Doe", givenName: "John", familyName: "Doe"});

var request = navigator.mozContacts.save(contact);


request.onsuccess = function() {
   alert("Success saving contact. New contact ID: " + contact.id);
};


request.onerror = function() {
   alert("Error saving contact.");
};

Web Activities

Handle an activity

"activities": {
   "share": {
     "filters": {
       type: ["image/png", "image/gif"],
      }
     "href": "foo.html",
     "disposition": "window"
   }
}

Launch an activity

var a = new Activity({ name: "pick", data: { type: "image/png", multiple: false }});
a.onsuccess = function() { var image = a.result; doSomethingWithImage(image); });
a.onerror = function() { alert("Failure when trying to pick an image!"); });

Push Notifications

  • No Polling
  • App registers for push endpoint
  • Application server can make HTTP request to push endpoint
  • No Payload
  • If app is not running it will be started in background
https://hacks.mozilla.org/2013/07/dont-miss-out-on-the-real-time-fun-use-firefox-os-push-notifications/

Writing Firefox OS Apps

  • https://github.com/robnyman/Firefox-OS-Boilerplate-App
  • Write an app in HTML5, CSS3, JavaScript
  • Make use of Open Web APIs
  • Make sure you have a manifest.webapp
  • Provide an offline.appcache file
  • Submit your app: https://marketplace.firefox.com/developers/submit/
